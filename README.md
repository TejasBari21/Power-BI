# Superstore Sales Dashboard

## Project Overview
This project is a **Power BI dashboard** created to analyze and visualize Superstore sales data.  
It helps in understanding sales performance, profit trends, customer segments, and regional performance through interactive visual reports.

## Features
- Sales analysis by **Region**
- Profit analysis by **Category** and **Sub-Category**
- Customer segment performance
- Monthly and yearly sales trends
- Interactive filters and slicers
- Easy-to-understand visual dashboard

## Tools Used
- **Power BI Desktop**
- **Superstore Dataset**
- Data cleaning and transformation using **Power Query**
- Data modeling using **DAX**

## File Included
- `Superstore Sales Dashboard.pbix` → Main Power BI project file

## Dashboard Insights
The dashboard provides insights such as:
- Which region generates the highest sales
- Which category gives maximum profit
- Monthly sales growth patterns
- Comparison of different customer segments
- Loss-making products or categories

## How to Open the Project
1. Install **Power BI Desktop**
2. Download or clone this repository
3. Open the `.pbix` file in Power BI Desktop

## Purpose of the Project
The main purpose of this dashboard is to:
- Practice data visualization in Power BI
- Build an interactive business intelligence report
- Analyze sales and profit performance effectively
